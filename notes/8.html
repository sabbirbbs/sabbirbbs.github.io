Live USB Encrypted Persistence

26/01/2021

Intro

There are have several linux distribution out there that support live system inside a pendrive with persistence.And sometime i have to use them as a portable system in my pocket. That's the reason i have to also use encryption for security purpose.Here is how to do that without any explaination.

Command

    

  • sudo bash
    • 

  • fdisk /dev/sda(n,e*4,w)
    • 

  • fdisk -l
    • 

  • cryptsetup --verbose --verify-passphrase luksFormat /dev/sda3
    • 

  • cryptsetup luksOpen /dev/sda3 usb
    • 

  • mkfs.ext4 /dev/mapper/usb
    • 

  • e2label /dev/mapper/usb persistence
    • 

  • mkdir -p /mnt/usb
    • 

  • mount /dev/mapper/usb /mnt/usb
    • 

  • echo "/ union" > /mnt/usb/persistence.conf
    • 

  • umount /dev/mapper/usb
    • 

  • cryptsetup luksClose /dev/mapper/usb
    •
